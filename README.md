**AESTIMUS Consulting** es una plataforma de gestión integral diseñada para centralizar y optimizar el portafolio de proyectos de ingeniería y consultoría. El sistema permite estructurar la planificación, ejecución y seguimiento de proyectos mediante una plataforma digital integrada.

---

## 🎯 Objetivo del Sistema

El sistema tiene como propósito principal alinear los **objetivos estratégicos** de la consultora con la **ejecución operativa** de sus iniciativas, facilitando:

```
✅ Organización del trabajo en proyectos, entregables y tareas
✅ Asignación clara de responsabilidades
✅ Trazabilidad de actividades del equipo
✅ Monitoreo sistemático del progreso
✅ Medición de indicadores de desempeño (KPIs)
✅ Generación de reportes ejecutivos
```

| 📒 **Directorio** | Base de datos del equipo de trabajo | ✅ Activo |

## 📊 Métricas y Dashboard

El sistema incluye un **Dashboard de Métricas Globales** con indicadores visuales en tiempo real:

- 📈 **Curvas de avance** por proyecto (% de progreso)
- 🎯 **KPIs de desempeño** del portafolio
- 📅 **Control de plazos** y fechas críticas
- ⚠️ **Alertas de prioridad** (Urgente / Alta / Media / Baja)
- 📋 **Reportes ejecutivos** exportables
