<br/>

**AESTIMUS Consulting** es una plataforma de gestión integral diseñada para centralizar y optimizar el portafolio de proyectos de ingeniería, consultoría y **valorizaciones de obra**. El sistema integra flujos de trabajo automatizados con **n8n** para maximizar la eficiencia operativa del equipo.

---

## 🎯 Objetivo del Sistema

El sistema tiene como propósito principal alinear los **objetivos estratégicos** de la consultora con la **ejecución operativa** de sus iniciativas:

<div align="center">

| ✅ Capacidad | Descripción |
|:---|:---|
| 📁 Gestión de Proyectos | Organización del trabajo en proyectos, entregables y tareas |
| 💰 Valorizaciones | Control y seguimiento de valorizaciones de obra por avance |
| 👥 Responsabilidades | Asignación clara de roles y actividades del equipo |
| 🔍 Trazabilidad | Registro completo de actividades desarrolladas |
| 🤖 Automatización | Flujos automatizados con **n8n** para reportes y notificaciones |
| 📊 Indicadores | Medición de KPIs y generación de reportes ejecutivos |

</div>

## 🤖 Automatización con n8n

**n8n** es la herramienta de automatización de flujos de trabajo integrada en el sistema de AESTIMUS Consulting para:

- 📧 **Notificaciones automáticas** de hitos y plazos críticos de proyectos
- 💰 **Alertas de valorización** por corte de avance de obra
- 📊 **Generación automática** de reportes ejecutivos periódicos
- 🔄 **Sincronización** entre módulos del sistema
- 📱 **Integración** con canales de comunicación del equipo
